# MDTO – Metagegevens voor Duurzaam Toegankelijke Overheidsinformatie

MDTO-conform betekent dat een informatiesysteem metagegevens van informatieobjecten vastlegt volgens het MDTO-metagegevensschema en deze beschikbaar en uitwisselbaar maakt. Er is onderscheid tussen minimaal MDTO-conform (vastleggen, bekijken en exporteren) en volledig MDTO-conform (gestructureerde weergave en XML-import/export).
